# 1．はじめに
 Maker Fair Tokyoで手に入れたBluetoohとwifiが搭載されているNefry BTというマイコンを手に入れたので色々遊んでみています.
 そこで，AndroidアプリとNefry BTの間でBLEを用いて通信をできるようにしてみました．  
 そのため，私的なメモですが記録を残したいと思います．また随時，改良していきたいと考えています．
